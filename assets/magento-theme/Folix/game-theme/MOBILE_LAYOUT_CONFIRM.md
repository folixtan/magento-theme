- 颜色：需要符合主题

**PC端菜单**：
- 内容：首页、游戏、点卡充值、新闻、支持、关于我们（Folix自己的）
- 显示位置：顶部水平导航
- 移动端隐藏

**实现方式**：
- 在default.xml中渲染两次catalog.topnav
- 一次给移动端（name="mobile.catalog.topnav"）
- 一次给PC端（name="desktop.catalog.topnav"）
- 用CSS媒体查询控制显示/隐藏
- **不用JS复制**
