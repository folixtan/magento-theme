# Folix Game Theme - 变量映射说明

## 📌 设计原则

### 原生变量优先
- **颜色相关**：统一使用 Magento 原生变量（通过 `_theme.less` 覆盖值）
- **原因**：确保与 Magento 核心组件兼容，减少维护成本

### 自定义变量扩展
- **功能扩展**：渐变、阴影、过渡、圆角、z-index 使用自定义变量
- **原因**：Magento 原生变量不支持这些功能

## 📁 文件职责

### `_theme.less` - 覆盖原生变量
- ✅ 只包含硬编码修改的变量
- ✅ 修改 Magento 原生变量的值
- ❌ 不引用其他变量

### `_variables.less` - 定义主题专属变量
- ✅ 定义渐变、阴影、过渡等扩展变量
- ✅ 定义组件专属颜色（可选）
- ⚠️ 颜色变量仅用于特殊场景，优先使用原生变量

### `_extends/*.less` - 扩展样式
- ✅ 使用原生变量（颜色）
- ✅ 使用自定义变量（渐变、阴影、过渡）
- ❌ 不定义新变量

### `_module.less` - 模块样式
- ✅ 使用原生变量（颜色）
- ✅ 使用自定义变量（渐变、阴影、过渡）
- ❌ 不定义新变量

## 🔍 验证方法

### 1. 检查变量一致性
```bash
# 查找原型文件中是否还有自定义颜色变量
grep -r "@folix-primary\|@folix-secondary\|@folix-text-\|@folix-bg-\|@folix-border\|@folix-error" assets/magento-theme/Folix/game-theme/web/css/source/extends/
```

### 2. 检查原生变量值
```bash
# 查看 _theme.less 中的原生变量值
grep "@theme__color__primary\|@theme__color__secondary\|@primary__color" assets/magento-theme/Folix/game-theme/web/css/source/_theme.less
```

### 3. 编译验证
```bash
# 部署静态内容，检查是否有编译错误
php bin/magento setup:static-content:deploy --theme Folix/game-theme
```
