## 四、响应式断点

### Desktop (≥ 1024px)
- 侧边栏：240px 宽
- 产品网格：4列
- 产品卡片：宽度计算 `(100% - 6%) / 4`

### Tablet (768px - 1023px)
- 侧边栏：200px 宽
- 产品网格：3列
- 产品卡片：宽度计算 `(100% - 4%) / 3`

### Mobile (< 768px)
- 侧边栏：隐藏，点击 Filter 按钮弹出
- 产品网格：2列
- 工具栏：简化显示
- 产品名称：截断单行

### Small Mobile (< 480px)
- 产品网格：2列，间距减小
- 产品名称：12px
- 价格：14px
- Swatch：更紧凑

## 五、交互效果

### 5.1 产品卡片 Hover 动画
```less
.product-item-info {
    transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
    
    &:hover {
        transform: translateY(-6px);
        border-color: #4A90E2;
        box-shadow: 0 12px 32px rgba(74, 144, 226, 0.2);
    }
    
    .product-image-photo {
        transition: transform 0.3s ease;
    }
    
    &:hover .product-image-photo {
        transform: scale(1.05);
    }
}
```

### 5.2 Swatch 选择效果
```less
.swatch-option {
    transition: all 0.2s ease;
    
    &:hover {
        transform: scale(1.1);
        border-color: #4A90E2;
    }
    
    &.selected {
        border-color: #4A90E2;
        box-shadow: 0 0 0 2px rgba(74, 144, 226, 0.3);
    }
}
```

### 5.3 Filter 折叠动画
```less
.filter-options-content {
    max-height: 0;
    overflow: hidden;
    transition: max-height 0.3s ease;
    
    &.active {
        max-height: 500px;
    }
}
```
