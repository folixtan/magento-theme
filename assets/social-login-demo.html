Third-Party Login Flow Demo

1

Click "Login" button in header

The login dropdown will appear with email/password fields and social login options.

2

Click "Google" button

A popup will appear for Google account selection. After authorization, Google returns an ID token.

google.accounts.id.prompt()
3

Backend verifies token

Send token to: POST /social/login/google

Backend verifies with Google and checks if email exists in Magento.

4

Account binding decision

If email exists: Show bind modal (one-click bind or password verify)

If new user: Show registration confirmation with pre-filled info

5

Login successful

Session created, user redirected to previous page or homepage.
